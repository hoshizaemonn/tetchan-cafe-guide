Part 2: 6つの投稿パターン＆テンプレート

カフェオープン後の日常運用で使える投稿パターンと具体的なテンプレート

1

今日のおすすめ型

毎日 ストーリーズ + 週3回 フィード
参考アカウント: 丘の上のパン屋 @okanouenopanya ー 約5,000投稿の継続力。毎日コツコツ投稿でフォロワー5.4万人。リアルタイム発信の教科書。

「今日行ったら何が食べられるか」をリアルタイムで伝える。来店動機を毎日作り続ける最も基本的なパターン。

投稿タイミング別の例

朝投稿（8〜9時）

「おはようございます。今日のコーヒーはエチオピア イルガチェフェ。フルーティーな香りが最高です」
→ 出勤前の人が「帰りに寄ろう」と思う

昼投稿（11:30〜12時）

「本日のランチプレート完成しました。地元野菜たっぷりのワンプレート、数量限定です」
→ ランチ客をリアルタイムで獲得

夕方投稿（17〜18時）

「今日も残りわずか。自家製チーズケーキ、あと3個です」
→ 帰宅途中の人が立ち寄る動機に

2

リール動画型（シズル感）

週2〜3本
参考アカウント: 猿田彦珈琲 @sarutahikocoffee / NO COFFEE @nocoffee_ ー コーヒーのシズル感をリールで表現。世界観の統一が参考。

フォロワー以外に届く最大の武器がリール。15秒の短い動画で「食べたい」「飲みたい」を刺激する。

4つの撮影パターン

コーヒー抽出（15秒）

豆を挽く→お湯を注ぐ→ドリップの滴り→カップに注ぐ→湯気のアップ
BGM: トレンド音源（ジャズ系が相性良い）

ラテアート（15秒）

ミルクを注ぎ始める→アートが描かれる過程→完成→少し引きの画
BGM: おしゃれ系トレンド音源

スイーツ断面（15秒）

ケーキやサンドイッチの全体像→ナイフで切る→断面のアップ→フォークですくう
BGM: ASMR系 or ポップ系

調理過程（15秒）

材料の並び→調理のハイライト→盛り付け→完成品の俯瞰→テーブルに置く
BGM: テンポの良い音源

撮影のコツ

  • 自然光がベスト（午前10時〜午後3時の窓際）
  • 手ブレ防止にスマホ三脚を使う（100均でもOK）
  • 同じ角度・同じトーンで統一感を出す
  • トレンド音源を使うとリーチが3〜5倍に
  • 最初の1秒で目を引くカット（いきなり注ぐ瞬間から始めるなど）
  • 投稿時間: 平日12-13時/19-21時、土日10-11時/17-23時

リール用キャプションテンプレート

今日の一杯☕

○○産のスペシャルティコーヒーを
丁寧にハンドドリップ。

△△な香りが広がる瞬間が
一番好きな時間です。

一杯¥○○○〜
テイクアウトもできます。

📍香椎 / ○時〜○時

#福岡カフェ #香椎カフェ #ハンドドリップコーヒー #カフェ巡り #福岡グルメ
成功のコツ

15秒以内に収める。最初の1秒が勝負（スクロールを止めさせる）。音源選びは発見タブの人気リールから探す。週2〜3本を継続することが大事。

3

人・裏側型

週2〜3回
参考アカウント: スズノイエ @cafe.suzunoie / メゾン・ド・イリゼ @maison_de_iriser ー 店主の人柄と裏側を見せて親近感を醸成。

「この人に会いたい」「この店の雰囲気が好き」と思わせる投稿。初めての来店ハードルを下げ、リピーターを増やす。

投稿例

開店準備の様子

朝の仕込み風景、コーヒー豆の焙煎、テーブルセッティング。「今日もお客さんのために準備してます」という日常感。

家族の日常

居酒屋を営む家族がカフェも始めるストーリー。お子さんの反応、奥さんとのメニュー試作など、パーソナルな一面。

お客様との交流

常連さんとの会話（許可取得済み）、お客さんからのメッセージ紹介。「温かい店だな」と伝わる投稿。

完璧に見せようとしない。「頑張ってる姿」「素の表情」が一番共感を集める。ストーリーズは気軽に、フィードは少し編集して投稿。

4

口コミ・お客様の声型

週1回
参考アカウント: KABEAT @kabeat_official ー ハイライト「お客様の声」に口コミをまとめ、新規来店者の信頼獲得に成功。

第三者の声は自分で「美味しい」と言うより何倍も説得力がある。お客様の感想を活用して信頼を構築する。

投稿例

お客様の投稿をリポスト

お客様がタグ付けしてくれた投稿をストーリーズでリポスト。フィードにまとめ投稿も作る。「嬉しい感想をいただきました」の形式。

口コミの紹介投稿

いただいた口コミをスクリーンショットや引用形式でシェア。店側のコメントも添えて感謝を伝える。

「今行かないと損」という緊急性で即日来店を促す。ストーリーズのリアルタイム性を最大限に活用するパターン。

6

コンセプト世界観型 NEW

月2〜4本（渾身の1本を作り込む）

このパターンの目的:
カフェの世界観を15〜30秒の動画で体験させ、「この空間に行きたい」を一発で作る。フォロー判断を即座に決定させる最重要コンテンツ。
世界観リールは「保存」「シェア」されやすく、アルゴリズム上最も有利。1本の良い世界観リールが数万リーチし、数百フォロワーを獲得する可能性がある。

4つの構成パターン（15〜30秒）

パターンA: 空間没入型

ドアを開ける → 店内パン → 窓からの光 → コーヒーのアップ → 店主の笑顔

「お客さんの目線」で店内を体験する構成。最初のドアを開けるシーンで一気に引き込む。

パターンB: 一日の流れ型

朝の準備 → 開店 → お客さんの笑顔 → 夕方の店内 → 「今日もありがとう」

カフェの1日をダイジェストで見せる。時間の経過で「毎日こんな素敵な場所がある」と感じさせる。

パターンC: 五感刺激型

豆を挽く音 → 湯気 → ラテアート → ケーキの断面 → 一口目の表情

音・映像・質感で五感を刺激。ASMR的な要素も取り入れて「体験したい」欲求を最大化。

パターンD: ストーリーテリング型

テキストオーバーレイで想いを語りながら、空間の映像を流す

「なぜこのカフェを作ったのか」「どんな場所にしたいのか」を文字と映像で伝える。共感でフォローに繋げる。

コンセプト別の方向性例（ヒアリング後に確定）

和モダン系

障子の光、木の質感、抹茶ラテ、BGMは環境音。「静寂の中の一杯」という世界観。

北欧系

白い壁、植物、シンプルな器、BGMはアコースティック。「ヒュッゲ（居心地の良さ）」を体現。

レトロ喫茶系

暖色照明、銅のケトル、プリン、BGMはジャズ。「昭和の喫茶店を令和に」のノスタルジー。

居酒屋×カフェ融合系

夜の居酒屋 → 朝のカフェの変身、二面性ギャップ。「夜は居酒屋、昼はカフェ。同じ場所の2つの顔」。

撮影・編集のコツ

  • 1カットは2〜3秒以内。テンポよく切り替える
  • 手持ちよりスライダーやジンバルで滑らかに（安いものでOK）
  • BGMはトレンド音源 or カフェの雰囲気に合う楽曲を厳選
  • 色味・フィルターを全投稿で統一（世界観のブレを防ぐ）
  • テキストオーバーレイは最小限。映像で語らせる

なぜこのパターンが最重要なのか

世界観リールは1本で数万リーチ→数百フォロワー獲得の可能性がある。「おすすめ型」や「人・裏側型」がフォロワーとの関係を深めるのに対し、世界観リールは「新しいフォロワーを爆発的に獲得する」役割。全フェーズで常に投入する。月2本でも、渾身の1本を作り込むことが大事。

まとめ＆アクションステップ

今日からできることを整理

参考アカウントの活用方法

Step 1: フォローする

上記のアカウントから5〜10個をフォロー。特にスズノイエ、猿田彦珈琲、NO COFFEEは必須。

Step 2: 研究する

毎日投稿をチェックして「なぜこの投稿が良いのか」を分析。写真の撮り方、キャプションの書き方、リールの構成を観察。

Step 3: 抽出する

自分のカフェに活かせる要素をリストアップ。完全コピーではなく、「成功の理由」を理解して自分の個性と掛け合わせる。

Step 4: 実践する

6つのパターンのテンプレートを使って投稿開始。まずはパターン1（今日のおすすめ型）とパターン3（人・裏側型）から。

6つのパターンの使い分け

パターン 頻度 目的 形式
1. 今日のおすすめ型 毎日ストーリーズ + 週3フィード 即日来店 ストーリーズ / フィード
2. リール動画型 週2〜3本 新規リーチ拡大 リール
3. 人・裏側型 週2〜3回 信頼構築・親近感 ストーリーズ / リール
4. 口コミ・お客様の声型 週1回 信頼・社会的証明 ストーリーズ / フィード
5. 限定・今だけ型 毎日ストーリーズ 即時売上 ストーリーズ
6. コンセプト世界観型 ★ 月2〜4本 認知爆発・フォロー獲得 リール（最重要）

最も大事なこと

完璧を目指さず、まず投稿を始めること。最初は上手くなくても大丈夫。
上記の参考アカウントも、最初の投稿と今の投稿を比べると大きく成長している。

6つのパターンの優先順位:
まずは パターン1（今日のおすすめ） と パターン3（人・裏側） から始める。
慣れてきたら パターン2（リール） と パターン6（世界観リール） を追加。
お客さんが来始めたら パターン4（口コミ） と パターン5（限定） をフル稼働。
